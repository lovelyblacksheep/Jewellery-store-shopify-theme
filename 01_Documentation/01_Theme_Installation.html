7. Theme settings. Top


In order to change your store of currency, follow the below steps:
Go to Admin > Settings > General > Store currency

Now, to select payment gateway you need to follow the below steps:
Go to Admin > Settings > Payments > Choose third-party providers > bogus gateway

You can select any country of currency you want by following these steps:
Go to Admin > Settings > Payments > Choose third-party providers > Countries/regions > Add country/region

You need to select the default option Accounts are required Follow the below steps:
Go to Admin > Settings > Checkout > Customer accounts > Accounts are required
