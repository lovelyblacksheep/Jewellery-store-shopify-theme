Codezeel Configurations For Jeweelish Themes


Codezeel Theme Settings


This is step by step configuration of Jeweelish - responsive theme's all section. You can configure all sections available with Jeweelish - responsive theme using this document.


1. Header Top Settings Top

To configure Header Top Block from admin bar, go to Online Store > Themes > Customize theme > Sections > Header Top Block settings tab from admin bar.



Country/region : Enable the Country/region selector.

Language : Enable the language.

Color scheme : Select background color of header top

Headertop text : Set text of header top section.

Top padding : Set top padding of section

Bottom padding : Set bottom padding of section



2. Header Settings Top

To configure Header from admin bar, go to Online Store > Themes > Customize theme > Sections > Header tab from admin bar.



Desktop logo position : Select logo position for desktop.

Sticky Header : Select sticky header option you want.

Color scheme : Select background color of header.

Wishlist Title : Set the wishlist title you want.

Wishlist Link : Set the redirection link of wishlist page.

Compare Title : Set the compare title you want.

Compare Link : Set the redirection link of compare page.

Mobile logo position : Select logo position for mobile.

Bottom margin : Set bottom margin of section.

Top padding : Set top padding of section.

Bottom padding : Set bottom padding of section.



2.1 Logo settings

To configure Logo from admin bar, go to Online Store > Themes > Customize theme > Theme Setting > Logo Settings tab from admin bar.



Logo image : Upload logo image for your store.

Desktop logo width : Set desktop width of logo. you can specify logo width from this option (max-width should be 190px).
